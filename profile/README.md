# Nya Media

We provide a free-range media streaming service for VRChat. Unfortunately, much of our core remains closed-source for a variety of reasons.

Feel free to reach out via our linked email if you'd like to help out!

## Our Stack

- Website (SvelteKit, Serverless)
- Scraper (Puppeteer, CF Bypasser, Docker)
- M3U8 Proxy (JS, Serverless)
